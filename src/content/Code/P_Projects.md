# Python 打包 exe

- 首先在 Pycharm 中的终端执行
  `pip install auto-py-to-exe`
  这是一个第三方库

- 把第三方库添加进来
  setting --
  Project: 选项 --
  Python interpreter --
  加号 ➕ 搜索刚才安装的第三方库
  `auto-py-to-exe`
  找到后点击 install Package
  即可完成第三方库的安装

## 打包文件

auto-py-to-exe
