# Store

## 项目简介

- 基于 Spring Boot + Vue3 的购物商城练习项目

## 技术栈

- 前端：Element Plus / Pinia / Axios / Vue 3 / Vite / TypeScript
- 后端：Spring Boot / MyBatis Plus / MySQL / Redis / JWT

## 功能概览

- 用户注册、登录
- 商品浏览与搜索、商品详情
- 购物车、订单管理、支付、评论
- 浏览器指纹关联登录状态（如有）
