# printf 常用格式符

| 格式 | 含义 |
| --- | --- |
| `%d` | 有符号十进制整数 |
| `%u` | 无符号十进制整数 |
| `%o` | 八进制整数 |
| `%x` | 十六进制整数（小写） |
| `%c` | 单个字符 |
| `%s` | 字符串 |
| `%f` | 浮点数 |
| `%.2f` | 保留两位小数 |
| `%e` | 科学计数法 |
| `%p` | 指针地址 |

## 示例

```c
printf("%d %u %x %c %s %.2f\n", a, b, b, ch, str, pi);
printf("%p\n", (void *)&a);
```
