# 函数传参方式

## 1. 位置参数

```python
def greet(name, age, sex):
    print(f"名字叫{name}，年龄{age}，性别{sex}")

greet("小米", 12, "男")
```

## 2. 关键字参数

```python
def info(name, age):
    print(f"我的年龄为{age}，我的名字叫{name}")

info(age=12, name="小米")
```

## 3. 缺省参数

```python
def profile(name, sex, age=12):
    print(f"名字叫{name}，年龄{age}，性别{sex}")

profile(name="小米", sex="男")
```

## 4. 不定长参数 *args

```python
def total(*args):
    return sum(args)

total(1, 2, 3)
```

## 5. 关键字参数 **kwargs

```python
def show(**kwargs):
    return kwargs

show(name="小米", age=12, id=66600)
```

## 6. 函数作为参数

```python
def calc(x, y):
    return x + y

def run(fn):
    return fn(1, 2)

run(calc)
```

## 7. lambda 匿名函数

```python
run(lambda x, y: x + y)
```
