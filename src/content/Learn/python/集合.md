# 集合（set）

集合是无序且元素唯一的容器，不支持下标索引。

## 差集

```python
set1 = {1, 2, 3}
set2 = {1, 5, 6}

set1.difference_update(set2)
set3 = set1.difference(set2)
```

`difference_update` 会修改原集合，`difference` 会返回新集合。

## 并集

```python
set1 = {1, 2, 3}
set2 = {1, 5, 6}
set3 = set1.union(set2)
```

## 数量

```python
set1 = {1, 2, 3, 4, 5, 1, 2, 3}
num = len(set1)
```
