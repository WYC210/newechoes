# 容器基础

Python 常见容器：`list`、`tuple`、`set`、`dict`。

## 常用操作

- `for` 遍历容器（字典默认遍历 key）
- `len()` 获取元素数量
- `max()` / `min()` 获取最大值/最小值

## 类型转换

- `list()` 转列表
- `tuple()` 转元组
- `str()` 转字符串
- `set()` 转集合

## 示例

```python
data = [3, 1, 2]
len(data)
max(data)
list(set(data))
```
