# 字典基础

字典通过键值对存储数据，键必须唯一。

## 常用操作

- 读取：`d[key]` 或 `d.get(key)`
- 新增/修改：`d[key] = value`
- 删除：`d.pop(key)`、`d.clear()`
- 遍历：`d.keys()`、`d.values()`、`d.items()`

## 示例

```python
scores = {"老鼠": 23, "猫": 12, "TOM": 22}
scores["张信哲"] = 66
scores["老鼠"] = 669
scores.pop("猫")
```

`len(scores)` 可以获取字典元素数量。
