# 字符串访问与常用方法

## 索引与切片

```python
s = "Python"
s[0]
s[1:3]
s[:3]
s[3:]
s[::-1]
```

## 替换与分割

- `replace(old, new)`：返回替换后的新字符串
- `split(sep)`：按分隔符拆分为列表

## 去空白

- `strip()`：去首尾空白
- `strip(chars)`：去首尾指定字符集

## 统计

- `count(sub)`：统计子串出现次数

## 示例

```python
text = "  aaabbb  "
text.strip()
"12aaabbb21".strip("12")
"a,b,c".split(",")
```
