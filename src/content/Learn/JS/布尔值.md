# 布尔值

`true` 和 `false` 表示逻辑真/假。JS 还有“真假值（truthy/falsy）”的概念。

## 常见 falsy

- `false`
- `0`、`-0`
- `""`（空字符串）
- `null`、`undefined`
- `NaN`

## 示例

```js
Boolean(0);
Boolean("");
Boolean("0");
Boolean([]);
```

上面结果依次为 `false`、`false`、`true`、`true`。
