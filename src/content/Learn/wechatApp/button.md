# button 常用属性

## type 与 size

```html
<button type="primary">跳转第二页</button>
<button type="warn" size="mini">打赏</button>
<button type="warn" size="mini">点赞</button>
<button type="warn" size="mini">收藏</button>
```

## 其他属性

```html
<button plain>镂空按钮</button>
<button disabled>禁用按钮</button>
<button loading>加载中</button>
```

常见 `type`：`primary`、`default`、`warn`。
